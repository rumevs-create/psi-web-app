⚡ QUICK START GUIDE

PSI Management System | Cheat Sheet untuk Pengguna Baru

🔐 1. LOGIN

1. Buka URL aplikasi di browser
2. Klik "Login dengan Google"
3. Pilih akun email perusahaan
4. Selesai! Anda masuk Dashboard

➕ 2. TAMBAH EQUIPMENT

1. Dashboard → Klik "Tambah Equipment"
2. Isi form (ID, Nama, Area, Unit, Type, dll)
3. Pilih Kategori: 1 Critical atau 2 Standard
4. Klik "Simpan"

📥 3. IMPORT EXCEL (MASSAL)

1. Siapkan file Excel sesuai template
2. Dashboard → "Import Data"
3. Drag & drop file Excel
4. Periksa mapping kolom → "Import"
Tip: Jika error "Periods kosong", klik "Import Tetap" untuk melanjutkan.

✏️ 4. EDIT & UPDATE

1. Cari equipment di Dashboard
2. Klik icon ✏️ Edit
3. Ubah data yang diperlukan
4. Klik "Update"

🏷️ 5. KATEGORI EQUIPMENT

Kategori Nilai Contoh
EHS Critical 1 Flame Detector, Deluge, Relief Valve
Standard 2 Pressure Vessel, Level Gauge

📝 6. FIELD WAJIB DIISI

  • Equipment ID (unik)
  • Nama Equipment
  • Area & Unit
  • Type
  • Tag Number
  • Description
  • Action Plan
Opsional: Periods, Kustodian, PSI Progress

🛠️ 7. TROUBLESHOOTING CEPAT

Masalah Solusi
Tidak bisa login Pastikan email terdaftar, clear cache browser
Import Excel gagal Periksa format .xlsx, pastikan header sesuai template
"ID sudah ada" Gunakan menu Edit untuk update, bukan Tambah Baru
Loading terus Refresh (F5), periksa koneksi internet
Periods kosong Klik "Import Tetap" - equipment tetap diimport

⚙️ 8. ACTION PLAN

  • Calibration: Kalibrasi instrument
  • Verification: Verifikasi fungsi
  • Monitoring: Pemantauan rutin
  • Replace: Penggantian part

📅 9. PERIODS (OPSIONAL)

  • T/A Shut-down
  • Annually / Half-yearly
  • Quarterly / Monthly
  • Weekly / Per-shift
  • Real-time
