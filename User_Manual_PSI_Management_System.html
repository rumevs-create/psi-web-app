1. Pendahuluan

PSI Management System adalah aplikasi web untuk mengelola data Process Safety Information (Informasi Keselamatan Proses) di industri petrokimia, kilang, atau pabrik kimia lainnya.

Tujuan Aplikasi
  • Menyimpan dan mengelola data equipment safety secara terpusat
  • Memantau progress PSI (Process Safety Information) per equipment
  • Mengidentifikasi equipment yang termasuk kategori EHS Critical
  • Mempermudah audit dan compliance terhadap regulasi safety

System Requirements

  • Browser modern (Chrome, Firefox, Safari, Edge) versi terbaru
  • Koneksi internet stabil
  • Akun Google untuk login (email perusahaan)

3. Login & Logout

Cara Login

1 Buka aplikasi di browser dengan mengakses URL aplikasi PSI Management System
2 Klik tombol "Login dengan Google" di halaman login
3 Pilih akun Google perusahaan Anda (email korporat)
4 Verifikasi berhasil - Anda akan diarahkan ke Dashboard
Penting!

Gunakan selalu akun email perusahaan yang sudah terdaftar. Jika belum memiliki akses, hubungi Admin IT.

Cara Logout

1 Klik tombol "Logout" di pojok kanan atas navbar
2 Konfirmasi logout - Anda akan kembali ke halaman login

5. Tambah Equipment Baru

Untuk menambahkan equipment satu per satu:

1 Dari Dashboard, klik tombol "Tambah Equipment" di pojok kanan atas
2 Isi Form dengan data lengkap equipment:
  • Equipment ID: Kode unik equipment (contoh: 11PVV014)
  • Nama Equipment: Nama lengkap (contoh: V 1101)
  • Area: Lokasi area (CP, Utility, dll)
  • Unit: Nomor unit (1100, 1200, dll)
  • Type: Tipe equipment (Pressure, Level, Temperature, dll)
  • Tag Number: Nomor tag instrument
  • Description: Deskripsi fungsi equipment
  • Action Plan: Rencana tindakan (Calibration, Monitoring, dll)
  • Periods: Periode maintenance (T/A Shut-down, Annually, dll)
  • Kategori: Pilih EHS Critical atau Standard
  • Kustodian: Penanggung jawab equipment (opsional)
  • PSI Progress: Persentase progress (0-100%)
3 Klik "Simpan" untuk menyimpan data
Perhatian

Equipment ID harus unik! Sistem akan menolak jika ID sudah ada di database.

6. Import Data Massal

Fitur ini digunakan untuk mengimport banyak equipment sekaligus dari file Excel atau CSV.

Persiapan File Excel

1 Download Template dari halaman Import Data
2 Isi data sesuai format template. Kolom wajib:
  • equip_id, nama, area, unit, type
  • tag_number, description, action_plan
Kolom opsional: periods, kategori, kustodian, psi_progress
3 Simpan file dalam format .xlsx atau .csv

Proses Import

1 Buka halaman Import dari Dashboard → klik "Import Data"
2 Upload file dengan drag & drop atau klik "Pilih File"
3 Preview data - Sistem akan menampilkan 10 baris pertama
4 Mapping kolom - Pastikan kolom Excel sesuai dengan field database
5 Klik "Import ke Database" dan tunggu proses selesai
Fitur Cerdas Import
  • Auto-mapping: Sistem otomatis mendeteksi kolom jika nama header sesuai
  • Duplicate handling: Jika equip_id sudah ada, sistem akan UPDATE data lama
  • Validasi: Sistem memeriksa data kosong sebelum import
  • Progress bar: Pantau progress import real-time
Penanganan Error

Jika muncul error "Periods kosong" atau field lain:

  1. Periksa baris yang disebutkan di error message
  2. Beberapa equipment memang tidak punya period (contoh: Rupture disc)
  3. Klik "Import Tetap (Lewati Error)" untuk melanjutkan
  4. Baris dengan data wajib kosong akan di-skip, yang lain tetap diimport

7. Edit Equipment

Untuk mengubah data equipment yang sudah ada:

1 Cari equipment di Dashboard atau halaman list equipment
2 Klik icon edit pada equipment yang ingin diubah
3 Ubah data yang diperlukan di form edit
4 Klik "Update" untuk menyimpan perubahan
Catatan

Equipment ID tidak bisa diubah setelah dibuat. Jika perlu ganti ID, hapus equipment lama dan buat baru.

8. Lihat Detail Equipment

Untuk melihat informasi lengkap equipment:

1 Klik icon eye pada equipment di dashboard
2 Halaman detail akan menampilkan semua informasi equipment termasuk:
  • Data lengkap equipment
  • Status PSI Progress
  • History perubahan (jika ada)
  • Informasi pembuat dan tanggal dibuat

9. Kategori Equipment

Sistem mengklasifikasikan equipment menjadi 2 kategori:

Kategori Nilai Deskripsi Contoh
EHS Critical 1 Equipment yang berkaitan dengan safety kritis (kebakaran, ledakan, toxic) Flame Detector, Deluge System, Relief Valve
Standard 2 Equipment umum yang tidak termasuk kategori critical Pressure Vessel, Level Gauge, Temperature Sensor
Cara Menentukan Kategori
  • EHS Critical: Equipment yang kegagalannya bisa menyebabkan kecelakaan fatal, kebakaran, ledakan, atau pelepasan bahan berbahaya
  • Standard: Equipment operasional normal yang kegagalannya tidak berdampak langsung pada safety

10. Troubleshooting

Masalah Umum dan Solusi

  • Pastikan menggunakan email perusahaan yang sudah terdaftar
  • Clear cache browser dan coba lagi
  • Hubungi admin jika akun belum memiliki akses

  • Pastikan format file .xlsx atau .csv
  • Periksa apakah header kolom sesuai template
  • Pastikan tidak ada baris kosong di tengah file
  • Coba download template baru dan copy-paste data Anda

  • Equipment ID harus unik (tidak boleh sama)
  • Jika ingin update data, gunakan menu Edit bukan Tambah Baru
  • Untuk import: sistem akan otomatis update jika ID sudah ada

  • Periksa koneksi internet
  • Refresh halaman (F5)
  • Clear browser cache (Ctrl+Shift+Delete)
  • Coba buka di browser lain
